<h1 align="center">CSS FLEXBOX SIMULATOR</h1>

<br>

Aplicación para practicar, de manera simulada, el comportamiento del sistema de diseño FLEXBOX.

La aplicación no es perfecta, pero te puede ayudar a entender mejor las principales propiedades de este sistema de diseño:

- display: flex
- flex-direction
- justify-content
- align-items
- align-content
- flex-wrap

No son las únicas, pero sí las más básicos y fundamentales que hay que tener muy claras a la hora de estudiar CSS y HTML.

Para utilizarlo, simplemente abre el documento flexbox-simulator.html mediante cualquier navegador.
